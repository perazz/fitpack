Detailed Description

OOP wrapper for bivariate spline fitting on a gridded polar disc.

Provides fitpack_grid_polar, a derived type for fitting bicubic splines to data sampled on a polar grid over a disc of constant radius . The coordinate mapping is:

Continuity constraints at the origin and optional specification of the function value are supported. Uses the pogrid core routine.
