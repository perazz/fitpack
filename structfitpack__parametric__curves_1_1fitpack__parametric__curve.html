Detailed Description

Open parametric curve fitter in .

Fits a parametric spline curve through data points associated with strictly increasing parameter values . If no parameter values are provided, they are computed automatically from cumulative chord lengths.

See also
Dierckx, Ch. 9, §9.1 (pp. 199–212); parcur
